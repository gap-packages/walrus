2.1 Creating pregroups

2.1-1 PregroupByMultiplicationTable
‣ PregroupByMultiplicationTable( enams, table )( function )
‣ PregroupByMultiplicationTableNC( enams, table )( function )

Returns: A pregroup in multiplication table representation.

If enams is a list of element names, which can be arbitrary GAP objects, with the convention that enams[1] is the name of the identity element, and table is a square table of non-negative integers that is the multiplication table of a pregroup, then PregroupByMultiplicationTable and PregroupByMultiplicationTableNC return a pregroup in multiplication table representation.

By convention the elements of the pregroup are numbered [1..n] with 0 denoting an undefined product in the table.

The axioms for a pregroup are checked by PregroupByMultiplicationTable and not checked by PregroupByMultiplicationTableNC.

gap> pregroup := PregroupByMultiplicationTable( "1xyY",
>                                               [ [1,2,3,4]
>                                               , [2,1,0,0]
>                                               , [3,4,0,1]
>                                               , [4,0,1,3] ] );
<pregroup with 4 elements in table rep>

2.1-2 PregroupOfFreeGroup
‣ PregroupOfFreeGroup( rank )( function )

Returns: A pregroup of a free group of rank rank.

Returns a pregroup of a free group of finite rank rank. Pregroups of free groups are not stored in multiplication table representation for efficiency reasons.

gap> pregroup := PregroupOfFreeGroup(4);
<pregroup of free group of rank 4>

2.1-3 PregroupFromFreeProduct
‣ PregroupFromFreeProduct( G, H )( operation )

Returns: A pregroup in multiplication table representation.

If G and H are finite groups, then PregroupFromFreeProduct returns the pregroup consisting of the non-identity elements of G and H and an identity element. A product between two non-trivial elements is defined if and only if they are in the same group.

gap> pregroup := PregroupOfFreeProduct(SmallGroup(12,2), SmallGroup(24,4));
<pregroup with 35 elements in table rep>

2.2 Creating pregroup presentations

2.2-1 RandomPregroupPresentation
‣ RandomPregroupPresentation( pregroup, nrel, lrel )( function )

Returns: A pregroup presentation.

Generates a random pregroup presentation over the pregroup pregroup with nrel cyclically cancelled relators all of length lrel.

gap> pg := PregroupOfFreeProduct(CyclicGroup(2), CyclicGroup(3));
<pregroup with 4 elements in table rep>
gap> pr := RandomPregroupPresentation(pg, 5, 20);
<pregroup presentation with 3 generators and 5 relators>

2.2-2 TriangleGroup
‣ TriangleGroup( k, l, m )( function )

Returns: A pregroup presentation.

TriangleGroup returns a pregroup presentation of the \((k,l,m)\)-triangle group over the pregroup PregroupOfFreeProduct(CyclicGroup(k), CyclicGroup(l)).

2.2-3 TriSH
‣ TriSH( m, n )( function )

Returns: A pregroup presentation.

Returns a pregroup presentation of the quotient of the \((2,3,m)\) triangle group by the m-th power of the commutator of the generators of the two cyclic groups.
